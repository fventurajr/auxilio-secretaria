# auxilio-secretaria
# Controlar as viagens de trabalho

# Classes de negócios:
## Viajante
* nome
* empresa
* centro de custo
## Viagem
* tipo [ ida | volta ]
* cia aérea [ Azul | Gol | Latam | outra ]
* número do vôo
* cidade origem
* aeroporto de partida [ GIG | SDU | outro ]
* data / hora do voo
* aeroporto de chegada [ outro | GIG | SDU ]
* cidade destino
* status de viagem
* secretária responsável
## Hospedagem
* nome do hotel
* dias de hospedagem
