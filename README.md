# auxilio-secretaria
## Controlar as viagens de trabalho

Clases de negócios:
## Viajante
* matrícula
* nome
* empresa
* centro de custo
## Viagem
* tipo [ ida | volta ]
* cia aérea [ Azul | Gol | Latam | outra ]
* número do vôo
* cidade origem
* aeroporto de partida [ GIG | SDU | outro ]
* data / hora do voo
* aeroporto de chegada [ outro | GIG | outro ]
* cidade destino
* status de viagem
## Hospedagem
* nome do hotel
* dias de hospedagem
